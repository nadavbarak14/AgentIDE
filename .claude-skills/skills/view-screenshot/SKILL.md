# View Screenshot

Capture a screenshot of the current preview browser viewport and save it as a PNG image.

## Usage

```bash
./scripts/view-screenshot.sh
```

## Parameters

None.

## Examples

```bash
# Take a screenshot of the current preview
./scripts/view-screenshot.sh
```
