# View Record Stop

Stop the current preview browser recording and save the video as a WebM file. Returns the file path of the saved recording.

## Usage

```bash
./adyx.view-record-stop/scripts/adyx.view-record-stop.sh
```

## Parameters

None.

## Examples

```bash
# Stop recording and get the saved file path
./adyx.view-record-stop/scripts/adyx.view-record-stop.sh
```
