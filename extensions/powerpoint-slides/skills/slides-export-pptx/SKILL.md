# slides-export-pptx

Exports the current slide deck to a PowerPoint (.pptx) file. Each slide's HTML is captured as an image and embedded into the PPTX.

## Usage

```bash
./scripts/slides-export-pptx.sh <output-path>
```

## Parameters

- `output-path` (required): Path for the output .pptx file (e.g., "presentation.pptx")
