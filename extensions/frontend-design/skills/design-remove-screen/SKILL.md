# design-remove-screen

Removes a screen from the Frontend Design extension by name. The tab is removed and the next available screen is selected automatically.

## Usage

```bash
./scripts/design-remove-screen.sh <screen-name>
```

## Parameters

- `screen-name` (required): The name of the screen to remove
